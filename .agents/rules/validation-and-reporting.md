# Validasi dan Pelaporan

- Gunakan script aktual di `package.json` dan focused checks sesuai risiko.
- Jangan menyatakan check yang tidak dijalankan sebagai lulus.
- Status manual validation hanya `PASS`, `PARTIAL`, `NOT RUN`, atau `BLOCKED`, dengan scenario dan evidence aktual.
- Review diff dan accidental files sebelum selesai.
- Laporan akhir mengikuti kategori wajib di root `AGENTS.md`, termasuk failure dan limitation.
