# Arsip Dokumentasi Kamsiber dan Security Dashboard

Direktori ini berisi dokumentasi impor atau dokumentasi historis dari Kamsiber dan Security Dashboard. Berkas-berkas tersebut dipertahankan sementara untuk kebutuhan penelusuran asal dan konteks.

Dokumentasi dalam direktori ini bukan sumber otoritatif untuk BacaNgaji Adventure dan tidak boleh digunakan sebagai acuan untuk:

- arsitektur atau technology stack;
- kontrak API;
- desain database atau migration;
- aturan keamanan;
- status fitur; atau
- perintah setup dan pengembangan.

Gunakan entry point dan bukti implementasi aktif di root repository serta `docs/README.md` untuk dokumentasi BacaNgaji Adventure.
