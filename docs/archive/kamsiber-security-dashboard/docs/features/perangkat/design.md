# Design Spec: Perangkat

## Overview
The `Perangkat` feature manages endpoints and devices (Laptops, Workstations) similar to the Server asset management.

## Pages & Components

### 1. PerangkatDashboard
*   Overview of device inventory, compliance, and endpoint agent statuses.

### 2. PerangkatList
*   Data Table: Device Name, User (PIC), OS, Endpoint Status, Compliance.

### 3. PerangkatDetail & PerangkatForm
*   Follows the exact same layout patterns as the `Server` module.
*   Tabs: Overview (Hardware specs), Endpoint Agents, Vulnerability (VA) findings, Assignment History.
