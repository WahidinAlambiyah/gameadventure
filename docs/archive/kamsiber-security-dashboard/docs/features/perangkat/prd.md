## 1. Deskripsi Singkat

Modul Perangkat akan mengelola inventaris perangkat endpoint (laptop, workstation, perangkat mobile, dll.) beserta status keamanannya. Detail fitur akan didefinisikan setelah modul-modul utama selesai dikembangkan.

---

> [!NOTE]

> Modul ini direncanakan untuk dikembangkan setelah seluruh modul utama (General, HR, Technology, Integration, Operations) selesai. File `design.md` yang sudah ada di folder ini berisi spesifikasi awal UI dari iterasi sebelumnya.
