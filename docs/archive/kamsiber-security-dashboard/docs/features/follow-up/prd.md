# PRD: Follow Up

**Nama Fitur:** Follow Up
**Status:** `[Next Phase]`
**Pemilik:** @product-management
**Modul:** Operations (O4)

---

## 1. Deskripsi Singkat

Modul Follow Up mengelola tindak lanjut mitigasi untuk setiap temuan keamanan yang ditemukan dari operasi VA dan/atau Pentest terhadap suatu aplikasi. Setiap temuan memerlukan aksi mitigasi yang harus dipantau hingga selesai.

---

## 2. User Stories

- Sebagai **Operator**, saya ingin mengelola tindak lanjut mitigasi untuk setiap temuan dari VA/Pentest, agar proses remediasi dapat dipantau hingga selesai.

---

## 3. Ketergantungan Modul

- **VA (O2)**: Temuan dari operasi VA menjadi input follow up.
- **Pentest (O3)**: Temuan dari operasi pentest menjadi input follow up.
- **Aplikasi (T4)**: Follow up terkait dengan aplikasi tertentu.
