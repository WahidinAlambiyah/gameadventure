# PRD: VA (Vulnerability Assessment)

**Nama Fitur:** VA
**Status:** `[Next Phase]`
**Pemilik:** @product-management
**Modul:** Operations (O2)

---

## 1. Deskripsi Singkat

Modul VA mengelola operasi Vulnerability Assessment yang dilakukan terhadap aplikasi. Mencakup pencatatan jadwal VA, hasil temuan, tingkat keparahan, dan status penanganan.

---

## 2. User Stories

- Sebagai **Operator**, saya ingin mengelola operasi VA yang dilakukan terhadap aplikasi, agar hasil assessment terdokumentasi.

---

## 3. Ketergantungan Modul

- **Aplikasi (T4)**: VA dilakukan terhadap aplikasi.
- **Integration (I1)**: Hasil VA mungkin diimpor dari tools pihak ketiga (Nessus, dll).
- **Follow Up (O4)**: Temuan VA memerlukan tindak lanjut.
