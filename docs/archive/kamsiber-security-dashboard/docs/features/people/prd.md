# PRD: People

**Nama Fitur:** People
**Status:** `[Next Phase]`
**Pemilik:** @product-management
**Modul:** Human Resources (H3)

---

## 1. Deskripsi Singkat

Modul People menyediakan daftar detail sumber daya manusia yang terlibat dalam operasi keamanan siber. Modul ini akan mencakup informasi personel, sertifikasi yang dimiliki, peran, dan pemetaan ke unit organisasi.

---

## 2. User Stories

- Sebagai **Admin**, saya ingin mengelola daftar sumber daya manusia beserta informasi detail mereka, agar data personel keamanan selalu mutakhir.

---

> **Status: Next Phase**
> Detail persyaratan akan dilengkapi oleh developer yang ditugaskan. File `design.md` yang sudah ada di folder ini berisi spesifikasi awal UI.
