# PRD: Security Awareness

**Nama Fitur:** Security Awareness
**Status:** `[Next Phase]`
**Pemilik:** @product-management
**Modul:** Human Resources (H4)

---

## 1. Deskripsi Singkat

Modul Security Awareness mengelola program, event, kampanye awareness keamanan, serta scoring keamanan untuk aset dan personel. Termasuk di dalamnya pengelolaan phishing simulation, e-learning, dan berbagai jenis kampanye kesadaran keamanan siber.

---

## 2. User Stories

- Sebagai **Operator**, saya ingin mengelola program, event, dan kampanye awareness keamanan, agar partisipasi dan efektivitas awareness terpantau.
- Sebagai **Operator**, saya ingin melihat scoring keamanan per aset/personel, agar area risiko tinggi dapat diprioritaskan.
